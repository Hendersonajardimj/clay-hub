## Next Steps

### 1. Add Real Content

Replace the sample content with your actual work:

```bash
# Your photos
public/images/photos/your-photo.jpg

# Your videos
public/videos/your-video.mp4

# Your album art
public/images/releases/your-album-cover.jpg
```

See `GETTING-STARTED.md` for detailed instructions.

### 2. Test Locally

```bash
npm run dev
# Open http://localhost:4321
```

Or preview the production build:

```bash
npm run preview
# Open http://localhost:4321
```

### 3. Deploy to Cloudflare Pages

**Option A: GitHub (Recommended)**

1. Push to GitHub:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: CLAY Hub site"
   git branch -M main
   git remote add origin https://github.com/your-username/clay-hub.git
   git push -u origin main
   ```

2. Go to [Cloudflare Dashboard](https://dash.cloudflare.com)
3. Pages → Create project → Connect to Git
4. Build settings:
   - Build command: `npm run build`
   - Build output: `dist`
5. Deploy!

**Option B: Direct Upload**

```bash
npm run build
npx wrangler pages deploy dist
```

### 4. Custom Domain

After deployment, add your custom domain in Cloudflare Pages settings.

Update `astro.config.mjs`:

```js
export default defineConfig({
  site: 'https://your-actual-domain.com',
  ...
});
```
